* SRFI 48: Intermediate Format Strings

** by Ken Dickey

This repository hosts [[http://srfi.schemers.org/srfi-48/][SRFI 48]]: Intermediate Format Strings, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /final/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-48/srfi-48.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-48/][join the SRFI-48 mailing list]] and send your message there.
